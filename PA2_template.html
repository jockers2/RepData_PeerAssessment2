Data Processing

The Storm Data file is downloaded if a local copy does not already exists. The read.csv function can directly read the .csv file embedded in the downloaded file.

# download file (if it is necessary)
fileUrl <- "https://d396qusza40orc.cloudfront.net/repdata%2Fdata%2FStormData.csv.bz2"
localFilename <- "StormData.bz2"
if (!file.exists(localFilename)) {
    download.file(fileUrl, destfile = localFilename, mode = "wb")
} 

# read the csv file embedded in the .bz2 file
StormData <- read.csv(localFilename)

First, those variables that have no recorded injuries nor fatalities plus no crop or property damage are identified. This permits dropping slightly more than 70% of the data for this analyis.

Even in this subset of data, the fields are completed in a very inconsistent manner. It is the opinion of this analysis that refresher training be provided to those recording the data to encourage the data recorded in a more uniform manner. The data will become more trustworthy and require less conjecture to prepare analytic data set suitable for study.

Property and Crop damage entries with + or - in the exponent field have been imputed with an exponent of 0. Investigating the few cases where this occurs, the values for property and crop damage are probablly grossly underestimated. However as two fatalities occur in the single variable with - for the exponent, I prefer not to drop the variables.

# tidy up parts off the data

# drop variables that have no effect (no damage, no injuries, no fatalities)
zeroPropDmg <- StormData$PROPDMG %in% 0
zeroCropDmg <- StormData$CROPDMG %in% 0
zeroFatal <- StormData$FATALITIES %in% 0
zeroInjury <- StormData$INJURIES %in% 0

zeroEffect <- (zeroPropDmg & zeroCropDmg & zeroFatal & zeroInjury)

StormData <- StormData[!zeroEffect,]

expSubs <- list(
    c("K",3),
    c("M",6),
    c("^$",0),
    c("m",6),
    c("\\?",0),
    c("h",2),
    c("H",2),
    c("B",9),
    c("k",3),
    c("[+]",0),
    c("[-]",0))

StormData$NewPROPDMGEXP <- StormData$PROPDMGEXP
StormData$NewCROPDMGEXP <- StormData$CROPDMGEXP

for (i in expSubs) {
    StormData$NewPROPDMGEXP <- gsub(i[1],i[2],StormData$NewPROPDMGEXP)
    StormData$NewCROPDMGEXP <- gsub(i[1],i[2],StormData$NewCROPDMGEXP)
}

StormData$NewPROPDMGEXP <- as.numeric(StormData$NewPROPDMGEXP)
StormData$NewCROPDMGEXP <- as.numeric(StormData$NewCROPDMGEXP)
